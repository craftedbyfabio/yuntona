# Cursor

AI-native code editor. Same governance implications as Copilot for proprietary code.

## What It Does

An AI-native code editor built on VS Code with deep model integration. Provides code completion, multi-file editing, codebase-aware chat, and increasingly autonomous coding capabilities. Same underlying technology concerns as GitHub Copilot.
